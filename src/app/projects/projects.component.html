Angular 4 Recipe list Angular Udemy Project

Created a recipe App in Angular 4 with a list of recipes and a recipe details sections which enables a user to add the ingredients to a shopping list. The project uses basic email and password authentication using Firebase. When a user registers on Firebase they are then able to store their own recipes in the Firebase database. The user interface uses Bootstrap 3. The project uses Angular routing, Http, Services, rxjs template driven and reactive forms in Angular and an introduction to ngrx.

Project undertaken as part of Maximilian Scharzmuller's Angular 4 course on Udemy.

Angular 4 Shop front end Angular Udemy Project

Created a shop front end in Angular 4 with a set of categories for bread, fruit, vegetables and seasonings and spices. The app uses authentication with Google using angularfire2 and firebase and the data is stored in firebase. A user who isn't registered can select items and add them to a shopping cart. However, once they want to check out a user has to sign in with their Google account in order to save items to firebase. The app uses a combination of localstorage and firebase to store data. Once the user has logged in to their Google signin they can view a my orders page. The option exists in firebase to give a user adminstrator rights so they can update the product pages using an Angular form. It would be possible to expand the prototype to add a payment fulfillment system such as Stripe or Paypal.

Project undertaken as part of Mosh Hamedani's Angular 4 course on Udemy.
